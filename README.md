### Descrição
A aplicação se trata de um observatório 3D do sistema solar com animação. O observador pode mover a câmera entre os planetas para observar melhor o sistema.



### Movimento da câmera
|  Tecla	|Movimento		|
|:-------------:|---------------|
| 8 / 2         | Dolly	        |
| 4 / 6	        | Truck		| 
| a / d	        | Pan	        |
|w / s		|Tilt		|
|↑ / ↓		|Pedestal	|
|r		|Reset		|
| P ou espaço	|Pause		|
